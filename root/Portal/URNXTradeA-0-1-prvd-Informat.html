Quick start guide for solution providers

   

Let others know that you started working on the task. This is important to avoid two or more persons work on the same issue: Only the first one finished will usually get the pay. You also need to indicate how long it will take you approximately to finish the task.

This step is optional.

Sample message: 

I decided to take up the OpenOffice related
task that Alice described in the message with
the following URN:

  <urn:X-trade:task:
   from=alice@me.foo;date=2006-04-09:
   a.0.1>

I plan to be done by 2006-04-20.

- Bob

<this-X-trade:strt:
 from=bob@xy.bar;date=2006-04-10:
 task_from=alice@me.foo;task_date=2006-04-09;
 tild=2006-04-20:
 a.0.1>

Learn how to create such a message.
